# MYNT: Material You New Tab - Personal Setup

This directory contains the configuration file for **MYNT: Material You New Tab** browser extension.

## 📋 About MYNT

MYNT is a versatile browser extension that personalizes your New Tab page with:

- Integrated search with multiple engines
- Customizable themes and wallpapers
- Clock, weather, and greeting widgets
- Quick shortcuts and AI tools
- To-do list and bookmarks
- Support for 32+ languages

**Official Repository:** [prem-k-r/MaterialYouNewTab](https://github.com/prem-k-r/MaterialYouNewTab)

## 💾 Backup & Restore

This directory contains `config.json` which stores all your MYNT settings:

- Theme preferences
- Shortcuts and bookmarks
- To-do list items
- Weather location
- Greeting text
- AI tool preferences
- And more...

1. Open a new tab and click the **settings icon** (circle icon, bottom right)
2. Scroll to the **Settings** section
3. Click **Restore** to load settings, or **Backup** to save them
4. Select the `config.json` file (restore) or choose download location (backup)

## 📝 Notes

- **Privacy:** MYNT stores all data locally in your browser. No data is sent to external servers.
- **Sync:** Settings don't sync across browsers/devices. Use this config file to manually transfer.
- **Updates:** After extension updates, your settings should persist, but keeping a backup is recommended.
